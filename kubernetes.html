kubernetes

概述 k8s 中创建和管理的最小单元。 包含一个或多个 container 容器 一个 Pod 中的容器共享网络命名空间 Pod 是短暂的 实现机制 共享网络机制 docker container 之间是相互隔离的，要实现共享网络，需要实现容器在同一个 namespace 之下。 Pod 实现共享网络机制的过程： 创建 Pause 容器（根容器、info 容器） 创建业务容器 将创建的业务容器加入到根容器中 循环 2-3 共享存储机制 通过 Volumn 数据卷实现（持久化存储） 镜像拉取策略 IfNotPresent： 默认值，不存在才拉取 Always： 每次
